# Roblox-studio-on-mobile
I am going to run Roblox studio on phones but I am learning c and pls support because I am using something different

• this project is for the kids who want to start making a game 

• for the people who can't afford a pc or laptop 

## Why I'm doing this

I've tried everything. Winlator. VectrasVM. Custom Windows. Nothing worked.

So I'm building my own solution. No x86 emulation. No VM. Just translating Windows API calls to Android.

It's hard. I'm learning C as I go. But I won't stop.
